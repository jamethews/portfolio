# Jamie Mathews, MD — GitHub Pages Portfolio

Static portfolio website for Clinical AI + 3D Slicer roles.

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload everything in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select **main** and **/root**, then save.
6. Your website will appear at `https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`.

## Important note

Use only truthful 3D Slicer experience on applications. The included work-sample sections are structured for de-identified or synthetic/demo imaging examples.
